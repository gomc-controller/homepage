## The Future of Open Industrial Automation

GOMC is an open-source framework for industrial motion control and automation. Built on Linux with real-time capabilities, it is designed for transparency, determinism, and extensibility — giving engineers full visibility into every layer of the stack.

At its core, GOMC pairs a **Go host runtime** with high-performance **C real-time plugins**, combining the productivity of modern systems programming with the deterministic execution required by demanding motion applications.

GOMC supports **IEC 61131-3 Structured Text**, cross-compiled to native code, so control logic written in the industry-standard language runs at full speed — no interpreter, no hidden overhead. A flexible **Hardware Abstraction Layer (HAL)** routes signals cleanly between components, making it straightforward to integrate sensors, drives, and fieldbus networks.
